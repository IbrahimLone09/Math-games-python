## Number guesser (python)
A fun command-line guessing  game built with python.The computer randomly selects a number and your mission is to guess it in the fewest attempts possible.

Perfect for python beginners learning about loops,conditionals and random numbers.

## Features
- Random numbers generation
- User input validation
- Hints if your guess is too high or too low
- Counts number of guesses used 
- Simple and interactive terminal gameplay
