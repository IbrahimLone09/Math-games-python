## Number guesser (python)
A fun command-line guessing  game built with python.The computer randomly selects a number and your mission is to guess it in the fewest attempts possible.

Perfect for python beginners learning about loops,conditionals and random numbers.

## Features
- Random numbers generation
- User input validation
- Hints if your guess is too high or too low
- Counts number of guesses used 
- Simple and interactive terminal gameplay

## How the game works
1. The user enters the maximum number of range
2. The program generates a random number between 0 and that number
3. You keep guessing untill you find the correct number
4. The program tells you if your guess is:

- Below the number
- Above the number
- Correct

5. When you guess correctly,the game shows how many attempts you used

## Technologies used
- python 3
- random module
- loops
- conditional statements

## How to Run
Make sure python is installed

python number_guesser.py
